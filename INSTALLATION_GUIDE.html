Step 1: Install Node.js

  • Visit nodejs.org and download the LTS version
  • Run the installer with default settings
  • Make sure "Add to PATH" is checked
  • Restart your computer after installation
‚ö†Ô∏è Important: Choose the LTS (Long Term Support) version for stability. The installer should automatically add Node.js to your system PATH.

Verify Installation:

node --version npm --version

You should see version numbers like v18.17.0 and 9.6.7

Step 2: Install Git

  • Download Git from git-scm.com
  • Install with default settings
  • Choose "Git from the command line and also from 3rd-party software"

Verify Installation:

git --version

Step 3: Install PostgreSQL

  • Download PostgreSQL from postgresql.org
  • During installation, set a password for the 'postgres' user
  • Remember this password - you'll need it later!
  • Use default port 5432
  • Install all components including pgAdmin
üîë Critical: Write down the postgres user password somewhere safe. You'll need it to configure the application!

üì•Download and Setup

Step 4: Download the Application

1. Create a Projects Folder:

  • Windows: Create C:\Projects\
  • Mac/Linux: Create ~/Projects/

2. Open Command Prompt/Terminal:

  • Windows: Press Win + R, type cmd, press Enter
  • Mac: Press Cmd + Space, type terminal, press Enter
  • Linux: Press Ctrl + Alt + T

3. Navigate and Download:

# Navigate to your projects folder cd C:\Projects\ # Windows cd ~/Projects/ # Mac/Linux # Download the application git clone https://github.com/arv-qa/iManageUploader.git # Enter the application folder cd iManageUploader

Step 5: Install Dependencies

What are dependencies? These are additional code libraries that our application needs to function properly. Think of them as ingredients for a recipe!
npm install

This command will:

  • Download all required libraries
  • Take 2-5 minutes depending on your internet speed
  • Show lots of text scrolling (this is normal!)
  • Create a node_modules folder

Step 6: Configure the Database

1. Create the Database:

  • Open pgAdmin (installed with PostgreSQL)
  • Connect using your postgres password
  • Right-click "Databases" ‚Üí "Create" ‚Üí "Database"
  • Name: imanage_uploader
  • Click "Save"

2. Configure Connection:

  • In the application folder, copy .env.example to .env
  • Open .env in a text editor
  • Update the DATABASE_URL line:
DATABASE_URL=postgresql://postgres:YOUR_PASSWORD@localhost:5432/imanage_uploader
Replace YOUR_PASSWORD with the actual password you set during PostgreSQL installation!

3. Create Database Tables:

npm run db:push

üîßTroubleshooting

Common Issues and Solutions

Issue: "npm is not recognized"
Solution: Node.js wasn't installed properly. Reinstall Node.js and make sure "Add to PATH" is checked. Restart your computer.
Issue: "Database connection failed"
Solution: Check your .env file password, make sure PostgreSQL is running, and verify the database exists.
Issue: "Port 5000 is already in use"
Solution: Another application is using port 5000. Change PORT=3000 in your .env file.
