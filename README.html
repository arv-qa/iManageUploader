🚀 iManage Uploader

A modern, web-based bulk document upload tool for iManage Work

React 18 TypeScript Express.js Node.js PostgreSQL Tailwind CSS Vite Drizzle ORM

📋 Overview

iManage Uploader is a comprehensive web application designed to streamline bulk document uploads to iManage Work. Built with modern web technologies, it provides an intuitive interface for users to authenticate, select workspaces, configure metadata, and upload multiple documents efficiently.

✅ Production Ready: This application includes authentication, progress tracking, error handling, and a responsive UI suitable for enterprise environments.

🌟 Key Features

🔐 Secure Authentication

OAuth2-compatible login system with token management and session handling for iManage Work integration.

📁 Workspace Management

Dynamic workspace selection with metadata field configuration based on workspace requirements.

📤 Bulk Upload

Multi-file upload with drag-and-drop support, progress tracking, and retry functionality for failed uploads.

📊 Real-time Progress

Live upload progress monitoring with detailed status for each file and overall session progress.

🎨 Modern UI

Responsive design built with Tailwind CSS and Radix UI components for excellent user experience.

🔄 Error Recovery

Intelligent retry mechanisms for failed uploads with detailed error reporting and recovery options.

🏗️ Architecture

Frontend: React 18 with TypeScript, Vite for build tooling, TanStack Query for state management, and Tailwind CSS for styling.
Backend: Express.js server with TypeScript, Drizzle ORM for database operations, and Multer for file handling.
Database: PostgreSQL with Drizzle ORM for type-safe database operations and migrations.

🚀 Quick Start

Prerequisites

  • Node.js 18+ and npm
  • PostgreSQL database
  • iManage Work server access

Installation

# Clone the repository git clone https://github.com/your-username/iManageUploader.git cd iManageUploader # Install dependencies npm install # Set up environment variables cp .env.example .env # Edit .env with your database and iManage configuration # Run database migrations npm run db:push # Start development server npm run dev
🎉 Ready to go! The application will be available at http://localhost:5000

📖 Usage Guide

Step 1: Authentication

Enter your iManage server URL, username, and password to authenticate and establish a secure session.

Step 2: Select Workspace

Choose from available workspaces. The application will automatically load the metadata fields required for that workspace.

Step 3: Upload Files

Drag and drop files or click to select multiple documents. Configure metadata fields as required by your workspace.

Step 4: Monitor Progress

Track upload progress in real-time with detailed status for each file. Retry failed uploads as needed.
